# Rubik-s-Cube-Timer
This is a Rubik's Cube Timer created in Python with the Tkinter Module with CSV files. This is a fun and interactive front-end interface for timing solves with different Rubik's Cube puzzles. The Repo consists of an Old and Improved Version (both written in Python) for demonstration of Rubik's Cube Timer. The Timer also uses tables to display a set of times which users can save and load in specified sessions wherever possible and shows your PBs per session.

Preferably Later Versions of Python is recommended but you may need to install packages and modules such as CSV, Tkinter and Datetime to run the program.

# Features
* Tkinter Module Interface with Treeview for table and Messageboxes for displaying messages like in a typical interface
* Object Oriented Programming Approach by Creating Classes for Scramble and Timer
* CSV files (Both original and Encrypted) to store data for Times
* Validation and Verification for entering times and Penalties
* Rule Page for New Users
* Saving and Loading Files with FileDialog for each session
* Calculating Mean, Average and Standard Devication
* Stats for Solve Count, Mean, Standard Deviation and Average per session
* Random Font Generation
* Customisable Background and Foreground (based on users choice but used every time a user opens the screen)
